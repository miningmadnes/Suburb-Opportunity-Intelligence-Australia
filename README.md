# Suburb-Opportunity-Intelligence-Australia

## Problem
Businesses often struggle to identify suburbs where service demand is growing faster than supply.

## Objective
Identify SA2 regions in Australia where population growth indicators exceed service availability.

## Data Sources
- Construction permits
- Healthcare locations
- Education locations
- Railway stations
- Road infrastructure
- ABS SA2 data

## Method
Demand signals:
- construction permits
- Crime data

Supply signals:
- healthcare locations
- education locations
- petrol stations

Infrastructure:
- railway access
- road access

## Output
Opportunity score for each SA2 region.
